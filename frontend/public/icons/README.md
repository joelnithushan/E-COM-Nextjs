## Required Icons

- `icon-16x16.png` - Favicon
- `icon-32x32.png` - Favicon
- `icon-72x72.png` - Android
- `icon-96x96.png` - Android
- `icon-128x128.png` - Chrome
- `icon-144x144.png` - Android
- `icon-152x152.png` - iOS
- `icon-192x192.png` - Android (required)
- `icon-384x384.png` - Android
- `icon-512x512.png` - Android (required)

## Generation

You can generate these icons using:
1. Online tools like https://realfavicongenerator.net/
2. PWA Builder: https://www.pwabuilder.com/imageGenerator
3. ImageMagick or similar tools

## Design Guidelines

- Use a simple, recognizable logo
- Ensure icons are readable at small sizes
- Use high contrast colors
- Follow platform-specific guidelines (iOS, Android)
- Test on actual devices
